Web Exploitation

What is "Web Exploitation"?

Web Exploitation, also called a browser exploit, is a form of malicious code that is usually placed inside websites and takes advantage of flaws or vulnerabilities in an operating system or software and alter a user's browser settings unnoticed.

How Web Exploitation works:

Web Exploits can come in many forms, and can exploit many things inside a browser including JavaScript and HTML.

According to the IEEE, the user can protect themselves from these exploits by simply keeping their browsers up to date as well as by installing firewall software or anti-virus software in their computers that can detect malicious websites
