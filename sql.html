SQL (Structured Query Language) Attacks

What are SQL attacks?

Though there are a few definitions for this, the most common one in regards to Cybersecurity is that Social Engineering is the use of deception to manipulate individuals into divulging confidential or personal information that may be used for fraudulent purposes.

Examples of SQL Injection attacks:

According to Portswigger, SQL Injection attacks are when vulerabilities in SQL are exploited, generally in order to retrieve data. A couple of examples of this are that SQL querys can be modified to return additional results, thus giving attackers access to hidden data, or UNION attacks, where data is retrieved from database tables.
