Wireless hacking

What is "Wireless Hacking"?

Wireless Hacking is typically defined as attacks on wireless networks or access points in which hackers use a variety of methods in order to gain access to a network and monitor it. This can be done maliciously but there are also cases where it is used for ethical purposes such as wiretapping.

How Wireless Hacking is done

One way Wireless Hacking is done is through "packet sniffing", this can be done through hardware such as a networktap or through software such as Wireshark.

During the sniffing process, a hacker can obtain the network's SSID from which they then move on to cracking the password to the network which can be done through various softwares such as WEPcrack on Windows or Aircrack-ng on Linux

Another form of Wireless Hacking is using Denial of Service (DoS) attacks which involves flooding a network's travid to the point where connections to the network slow down or even close completely. This can be done using programs such as Plyoris or Low Orbit Ion Cannon which can be found online.
